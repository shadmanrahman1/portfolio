Manim Math Animator Chatbot (2025)

Live demonstration: "animate sin(x)" command result

Project Overview

An intelligent AI chatbot that transforms mathematical equations into stunning animated visualizations. Simply chat with natural language prompts like "animate sin(x)", "show me y = x²", or "visualize the derivative of cos(x)" and watch as the system generates professional mathematical animations using Manim and advanced language models.

Smart Features
  • Conversational Interface: Natural language processing for mathematical requests
  • Equation Intelligence: Understands complex functions like sin(x), cos(x), tan(x), polynomials, logarithms
  • Dynamic Animations: Creates smooth mathematical transformations and visualizations
  • Video Generation: Exports high-quality MP4 videos ready for presentations
  • Code Generation: Powered by DeepSeek Coder for precise Manim code creation
  • Mathematical Concepts: Visualizes calculus, algebra, trigonometry, and advanced topics

  • "animate sin(x) from 0 to 2π" ⭐ Demo Above
  • "show y = x² transformation"
  • "visualize e^x growth"
  • "derivative of cos(x)"
  • "plot log(x) vs ln(x)"
  • "animate Fourier series"
